# Tokens Service

Este projeto é um microsserviço construído com uma stack Elixir minimalista.

O objetivo desse projeto é contemplar os requisitos descritos em [BACKEND-CHALLENGE.md](BACKEND-CHALLENGE.md).

## Executando a aplicação em ambiente local

Pré-requisitos:
- Elixir e Erlang instalados
- Para instalar siga [essa doc](https://elixir-lang.org/install.html).

Para iniciar o servidor:

* Execute `mix setup` para fazer qualquer setup necessário, incluindo a instalação de dependências
* Execute `mix start` para iniciar o servidor

A aplicação deve subir no endereço [`localhost:4000`](http://localhost:4000).
